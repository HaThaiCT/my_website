## Explanation

In this problem, we have `n` movies and their starting and ending times. We need to find out how many maximum movies we can watch, if we watch each movie completely.

We can do this by sorting all movies by their ending times. Then we'll watch movies that end the earliest. This way we'll be able to watch the maximum movies.

## Code

<CodeTabs langs={["Python", "Rust", "C++"]}>

```python
import sys
input = sys.stdin.readline

movies = []

for _ in range(int(input())):
    a, b = map(int, input().split())
    movies.append((b, a))
movies.sort()

count = 0
last = 0
for b, a in movies:
    if a >= last:
        count += 1
        last = b

print(count)
```

```rust
use std::io;

fn main() {
    let mut movies = vec![];
    for _ in 0..scan::<usize>() {
        let v = scan_vec::<usize>();
        movies.push((v[0], v[1]));
    }
    movies.sort_by_key(|&(_, e)| e);

    let mut count = 0;
    let mut last = 0;
    for (a, b) in movies {
        if a >= last {
            count += 1;
            last = b;
        }
    }

    println!("{}", count);
}

fn scan<T: std::str::FromStr>() -> T {
    let mut inp = String::new();
    io::stdin().read_line(&mut inp).unwrap();
    inp.trim().parse().ok().unwrap()
}

fn scan_vec<T: std::str::FromStr>() -> Vec<T> {
    let mut inp = String::new();
    io::stdin().read_line(&mut inp).unwrap();
    inp.trim()
        .split_whitespace()
        .map(|e| e.parse().ok().unwrap())
        .collect()
}
```

```cpp
#include <bits/stdc++.h>
using namespace std;

int main () {
    vector<pair<int, int>> movies;
    int n;
    cin >> n;

    while (n--) {
        int a, b;
        cin >> a >> b;
        movies.push_back({a, b});
    }
    sort(movies.begin(), movies.end(), [](pair<int, int> a, pair<int, int> b) {
        return a.second < b.second;
    });

    int count = 0, last = 0;
    for (auto movie : movies) {
        if (movie.first >= last) {
            count++;
            last = movie.second;
        }
    }

    cout << count << endl;
}
```
