## Problem Explanation

<hr />

Here, we are provided `n - 1` distinct numbers from **1 to n (inclusive)**.\
Our task is to find the missing number from the sequence **1, 2, 3, ... n**.

<br />

This can be done finding the sum of `n` numbers using `(n * (n + 1)) / 2` and subtracting the sum of the given numbers from it. 

## Example

<hr />

For numbers `1, 2, 3, 4, 5`, if the missing number was `2`. Using our method would be following these steps:

<br />

```python
sum_total = 1 + 2 + 3 + 4 + 5
sum_given = 1 + 3 + 4 + 5

ans = sum_total - sum_given = 2
```

```rust
use std::io::stdin;

fn main() {
    let n: usize = scan_str().trim().parse().unwrap();
    let s: usize = scan_str()
        .trim()
        .split_whitespace()
        .map(|x| x.trim().parse::<usize>().unwrap())
        .sum();
    println!("{}", (n * (n + 1)) / 2 - s);
}

fn scan_str() -> String {
    let mut s = String::new();
    stdin().read_line(&mut s).unwrap();
    s
}
```

```cpp
#include <bits/stdc++.h>
using namespace std;

int main () {
    int n;
    cin >> n;

    long long s = 0;
    for (int i = 0; i < n - 1; i++) {
        int x;
        cin >> x;

        s += x;
    }

    cout << ((1LL * n * (n + 1)) / 2 - s) << endl;
    return 0;
}
```
