## Explanation

<hr />

Here, we are given a number `n` and we need to apply the following operations on it:

- **number is odd?** : multiply by 3 and add 1
- **number is even?** : divide by 2

And we need to apply those until the number `n` becomes `1`. <br />

This can be solved by a simple `while` loop and implementing the operations.

## Code

<hr />

<CodeTabs langs={["Python", "C++", "Rust"]} >

```python
n = int(input())

while n > 1:
    print(n, end=" ")
    n = (n * 3 + 1) if n % 2 else (n // 2)

print(1)
```

```cpp
#include <bits/stdc++.h>
using namespace std;

int main () {
    long long n;
    cin >> n;

    while (n > 1) {
        cout << n << " ";
        n = (n % 2) ? (n * 3 + 1) : (n / 2);
    }

    cout << 1 << endl;
    return 0;
}
```

```rust
use std::io;

fn main() {
    let mut s = String::new();
    io::stdin().read_line(&mut s).unwrap();
  
    let mut n: usize = s.trim().parse().unwrap();

    while n > 1 {
        print!("{n} ");
        if n % 2 == 1 {
            n = 3 * n + 1;
        } else {
            n /= 2;
        }
    }

    println!("1");
}
```
